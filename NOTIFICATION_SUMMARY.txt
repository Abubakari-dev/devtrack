3. PROJECT ALERTS (📋 High Priority)

   - Sound: project_alert.wav (or system default)
   - Vibration: [0, 300, 200, 300]

4. TASK REMINDERS (⏰ High Priority)
   - Task Reminders
   - Subtask Alerts

   - Sound: reminder_tone.wav (or system default)

If you want custom sounds:
1. Download free sounds from notificationsounds.com
2. Create android/app/src/main/res/raw/ folder
3. Add sound files (lowercase, no spaces)
4. For iOS: Add sounds through Xcode
5. Rebuild app

Otherwise, you're all set! The system default sounds will play
automatically and sound like SMS notifications.
